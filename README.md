# 💸 Cloud-Based Expense Tracker

## 📌 Project Overview
This project is a simple web-based expense tracker hosted on Microsoft Azure.  
It allows users to add and view their expenses in a clean interface.

The project demonstrates the use of cloud computing services including deployment, storage, and automation.

---

## 🚀 Features
- Add expenses (amount & category)
- View list of expenses
- Simple and user-friendly interface
- Deployed on Azure Static Web Apps

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Microsoft Azure (Static Web Apps)
- GitHub (Version Control)

---

## ☁️ Azure Services Used
- Azure Static Web Apps → Hosting the frontend
- Azure SQL Database → For structured data storage (conceptual)
- Azure Blob Storage → For file storage (conceptual)
- Azure Logic Apps → For automation (conceptual)
