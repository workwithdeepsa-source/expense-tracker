# 💸 Cloud-Based Expense Tracker

## 📌 Project Overview
This project is a web-based expense tracker that allows users to add, view, and analyze their expenses.  
It is designed using HTML, CSS, and JavaScript, and deployed using GitHub Pages.

The project demonstrates the practical use of cloud computing concepts such as deployment, storage, and automation.

---

## 🚀 Features
- Add expenses (amount & category)
- View list of expenses
- Display total spending
- Pie chart visualization of expenses
- Data stored using browser localStorage
- Responsive and aesthetic UI

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js (for graphs)
- GitHub (Version Control & Deployment)

---

## ☁️ Deployment
The project is deployed using GitHub Pages and is accessible online.

## ☁️ Azure Services (Conceptual Implementation)
The following Azure services were explored as part of the project:

- Microsoft Azure Static Web Apps → for hosting (attempted)
- Azure SQL Database → for structured data storage
- Azure Blob Storage → for file storage
- Azure Logic Apps → for automation workflows

Due to subscription policy restrictions, deployment was completed using GitHub Pages instead of Azure Static Web Apps.
